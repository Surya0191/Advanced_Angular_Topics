DOM Sanitizer Demo

This example demonstrates how Angular's DOM Sanitizer handles potentially unsafe HTML. Without sanitization, Angular removes scripts to prevent security risks. With sanitization, we manually trust the HTML using bypassSecurityTrustHtml().

❌ Without DOM Sanitizer

Angular automatically **removes unsafe content** such as scripts to **prevent XSS attacks**.

✅ With DOM Sanitizer

Here, we use sanitizer.bypassSecurityTrustHtml() to **manually trust** the content. This **keeps the original HTML intact**, but can be dangerous if not handled carefully.
